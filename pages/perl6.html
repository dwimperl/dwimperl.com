=title Perl 6 for Linux The Rakudo team provides a source code distribution that you can download.

This package contains Perl 6 Rakudo Star 2014.12.1 compiled for Linux specifically to be used with Travis-CI but it can be used elsewhere too. The whole Perl 6 Maven runs on it. AFAIK the only limitation is that it needs to be installed in the /user/travis directory as path-es are hard-coded. 

$ sudo mkdir /home/travis
$ sudo chown foobar.foobar /home/travis
$ cd /home/travis
$ wget http://dwimperl.com/download/rakudo-2014-12-1-2.tar.gz
$ tar xzf rakudo-2014-12-1-2.tar.gz
$ cd

$ /home/travis/rakudo-2014-12-1-2/bin/perl6 -v
This is perl6 version 2014.12 built on MoarVM version 2014.12
You can add /home/travis/rakudo-2014-12-1-2/bin to your PATH environment variable 
export PATH=/home/travis/rakudo-2014-12-1-2/bin:$PATH
and then you don't need to type in the full path every time you run a script: 
$ which perl6
/home/travis/rakudo-2014-12-1-2/bin/perl6

$ perl6 -v
This is perl6 version 2014.12 built on MoarVM version 2014.12

The Rakudo team provides two msi installers for Windows with two different backends compiled in. You can download those from the Rakudo web site.
