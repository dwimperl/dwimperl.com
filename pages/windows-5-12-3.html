=title DWIM Perl for Windows - Perl 5.12.3

You are looking at an earlier release.
Unless you are an archaeologist, you probably want to check out the latest version of DWIM Perl for Windows.

Windows

Dwimperl-5.12.3.0-v6.exe, released on 2012.02.06.

It contains:

  • Strawberry Perl 5.12.3.0
  • Padre, the Perl IDE 0.94
  • Module::Version 0.12 so you can use mversion to check which version of each module you have.
  • Moose 2.0205, the post modern Object System
  • Perl::Critic 1.116, to police yourself.
  • Perl::Tidy 20101217, to keep your code nice
  • SQLite 1.35, to hold your data tight.
  • MySQL 4.018 and PostgreSQL 2.18.0 drivers
  • Lots of additional modules... (see details in the README file.)
