=title DWIM Perl for Linux server

This is a plain perl distribution of perl 5.18.1 built on CentOS 6.4 x64. It is a very experimental version - use it with caution.

Instructions:
  • Get the file and upload it to your server.
  • Unzip using tar xzf dwimperl-linux-5.18.1-1-x86_64.tar.gz
  • Run dwimperl-linux-5.18.1-1-x86_64/perl/bin/perl -V
  • Check out the README.txt that comes with the distribution
